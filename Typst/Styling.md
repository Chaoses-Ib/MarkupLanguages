## Dark mode

```typst
#set page(fill: black)
#set text(fill: white)
```
GitHub:
```typst
#set text(white)
#set page(fill: "#0d1117")
```
```typst
#let text-color = black
#let background-color = white
#if sys.inputs.at("theme", default: "light") == "dark" {
  text-color = rgb(240, 246, 252)
  background-color = rgb("#0d1117")
}
#set text(text-color)
#set page(fill: background-color)
```
